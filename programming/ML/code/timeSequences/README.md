### Исследования на стационарность
- Стационарный ряд - это временной ряд, статистические свойства которого не меняются со временем.
- Если ADF < критического значения → ряд СТАЦИОНАРЕН
- Если ADF > критического значения → ряд НЕСТАЦИОНАРЕН
- p-value = а - вероятность а%, что мы ошибочно отвергаем нестационарность
- Уровень надёжность 0.95, то есть значение p-value должно быть меньше 0.05, чтобы принять, что ряд стационарен

### Модели для временных рядов
Реализованы 2 наивные модели, которые предсказывают значения исходя из среднего и последнего
И модели Хольта-Уинтерса и SARIMA. Можно видеть, что Специальные модели справляются не лучше наивных,
что подтверждает факт непредсказуемости курсов валют.

Сверяю показатели ориентируясь на:
- MSE (Mean Squared Error) - Средняя квадратичная ошибка: MSE = (1/n) * Σ(y_i - ŷ_i)^2
- MAE (Mean Absolute Error) - Средняя абсолютная ошибка: MAE = (1/n) * Σ|y_i - ŷ_i|
- RMSE (Root Mean Squared Error): RMSE = √MSE
- MAPE (Mean Absolute Percentage Error): MAPE = (100%/n) * Σ|(y_i - ŷ_i)/y_i|
