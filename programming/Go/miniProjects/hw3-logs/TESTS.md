### Тестирование проекта

#### Blackbox тесты

В проекте так же есть ряд тестов, реализованных по методике **"черного ящика"**.

Данные тесты проверяют, что ваша реализация соответсвует описанному в задании техническому заданию.
С самими тестами можно ознакомится подробнее в файле `scripts/run_acceptance_tests.sh`.

Для каждого теста дано краткое описание. Большая часть тестов - негативные, и проверяют корректность работы
вашей программы в случае передачи невалидных данных.

Помимо них, так же есть один позитивный тест-кейс, который проверяет валидность собираемой статистики. Тестовые данные
для само-проверки можно найти в директории `scripts/data/input/logs`.
